Methodology

How this was built

Sources

  • YouTube channel, Videos, and Live tabs read from public pages on July 21, 2026; 28-day growth from socialcounts.org (YouTube Data API–backed).
  • Twitch metrics from TwitchTracker (30-day hours, average/peak concurrent viewers, per-stream log) and streamscharts (follower total), same date.
  • X and Instagram follower counts from public profile snippets — approximate, marked as such.

Definitions

Avg concurrent viewers (CCV)
The average number of people watching the Twitch stream at any moment while it was live — a live-attendance metric.
VOD views
YouTube views on the stream's archived recording, which continue accruing after the stream ends. CCV and VOD views are different units and are never summed.
Matched stream
A stream where the TwitchTracker log entry and the YouTube Live VOD align 1:1 by date. 8 of the last ~16 streams matched confidently; unmatched ones (multi-stream days, possible single-platform streams) are excluded from the simulcast view but present in the VOD chart.
Maturity window
Items under 7 days old are still accruing views — drawn lighter and excluded from medians, same convention as the companion Barstool Show Analytics dashboard.

Simulcast caveats

  • True live reach = Twitch CCV + YouTube live concurrents, but YouTube doesn't expose live concurrents publicly — so the live column is a floor, not a total.
  • Twitch VODs expire under standard retention, so the YouTube archive is where long-tail value accumulates — one reason VOD-optimized titles/thumbnails matter even for live-first content.
  • Simulcasting splits the live chat/community across two rooms; per-platform chat activity would be the next metric to capture.

Limitations

  • One snapshot; velocity and decay need repeated captures. The three most recent VODs (Champs weekend) are still accruing.
  • Stream-to-VOD pairing is by date; a mismatch would attach the wrong CCV to a VOD. Only confident 1:1 matches are shown.
  • YouTube Shorts, clips channels, TikTok, and podcast audio are out of scope.

Productionizing this

  • Twitch Helix API (streams, followers) + YouTube Data API on a nightly schedule joins the two platforms on stream date/time properly — no manual matching.
  • That unlocks live-vs-VOD decay curves per stream, CCV-by-hour heatmaps (best days/times to stream), and event-weekend lift measurement against baseline weeks.
